Sauvegarde & Synchronisation

Comment fonctionnent les données ?
Les données (membres, événements, offres, articles) sont stockées dans le navigateur de cet appareil. Si vous vous connectez depuis un autre PC ou navigateur, les données ne seront pas présentes.
Solution : exportez régulièrement la sauvegarde et importez-la sur chaque appareil des administrateurs.

Étape 1 — Exporter la sauvegarde

Cliquez sur le bouton ci-dessous pour télécharger un fichier .json contenant toutes les données du site (membres, événements, offres, articles, journal). Conservez ce fichier en lieu sûr.

Étape 2 — Importer sur un autre appareil

Pour synchroniser un autre appareil, connectez-vous en tant que Maître, ouvrez cette section, puis sélectionnez le fichier .json exporté précédemment.

Guide : synchroniser les données entre plusieurs adminstrateurs
  1. Administrateur principal (ce poste) : Cliquez Exporter la sauvegarde. Envoyez le fichier .json par e-mail ou clé USB aux autres admins.
  2. Autres admins : Connectez-vous en tant que Maître sur leur appareil, ouvrez Sauvegarde, sélectionnez le fichier reçu, cliquez Remplacer toutes les données.
  3. Convention : Seul un poste fait foi. Exportez avant chaque séance d'administration et importez sur tous les autres postes.
  4. Conseil : Programmez une exportation mensuelle automatisée via le rappel ci-dessous.

Gestion des versions & mises à jour

À quoi sert cette page ?
Elle sert à publier les versions Windows, Android et Web, et à contrôler l'occupation des données du site côté navigateur.
Ce que signifie la taille
Les tailles .exe et .apk décrivent vos applications. La taille du site affichée plus bas mesure les données synchronisées du site dans le navigateur.
Ce qui n'est pas mesurable ici
L'espace restant exact sur GitHub Pages ou Google Drive n'est pas exposé au navigateur. La page affiche donc un quota navigateur estimé.
Comment fonctionne le système de mise à jour ?
Le fichier assets/version.json est la source de vérité pour les versions des applications. Quand vous publiez une nouvelle version de l'.exe ou de l'.apk, mettez à jour ce formulaire et téléchargez le nouveau version.json à déposer sur le serveur. Les utilisateurs verront automatiquement une bannière de notification lors de leur prochaine visite.

Guides d'installation pas à pas

Installation sur PC Windows

  1. Téléchargez le fichier en cliquant sur « Télécharger (.exe) » ci-dessus. Le fichier amicale-attt-setup.exe apparaît dans votre dossier Téléchargements.
  2. Ouvrez le fichier. Si Windows affiche une fenêtre SmartScreen :
    • Cliquez sur « Informations complémentaires »
    • Puis sur « Exécuter quand même »
  3. Suivez l'assistant d'installation, acceptez la licence, choisissez le dossier, cliquez Installer.
  4. Attendez la fin de l'installation — un raccourci est créé sur le Bureau et dans le menu Démarrer.
  5. Lancez l'application depuis le raccourci Bureau ou le menu Démarrer.

Vérification de sécurité

Clic droit sur le fichier .exe → Propriétés → onglet Signatures numériques pour vérifier l'authenticité.

Configuration requise

  • Windows 10 / 11 (64 bits)
  • Processeur 1 GHz ou plus
  • 4 Go de RAM
  • 200 Mo d'espace disque
  • Résolution 1024×768 min.

Installation sur Android (Smartphone & Tablette)

Le même fichier .apk fonctionne sur smartphone ET tablette Android. L'interface s'adapte automatiquement à la taille de l'écran.
  1. Téléchargez le fichier amicale-attt.apk depuis le bouton ci-dessus.
  2. Autorisez les sources inconnues (une seule fois) :
    • Allez dans Paramètres → Sécurité
    • Activez « Sources inconnues » ou « Installer des apps inconnues »
    • Sélectionnez votre navigateur et autorisez
  3. Ouvrez le fichier via la notification de téléchargement ou votre gestionnaire de fichiers.
  4. Tapotez « Installer » et attendez la fin.
  5. Lancez l'application depuis votre écran d'accueil.

Pourquoi pas le Play Store ?

Distribution directe (APK) pour éviter les coûts de publication. L'application est 100 % sécurisée et produite par l'équipe ATTT.

Compatibilité

  • Android 8.0 (Oreo) ou supérieur
  • Smartphones Android
  • Tablettes Android
  • 50 Mo d'espace libre

Installation sur iPhone, iPad, Mac, Linux, Chromebook…

iPhone & iPad (Safari)

  1. Ouvrez Safari et naviguez sur le site
  2. Tapotez l'icône Partager
  3. Tapotez « Sur l'écran d'accueil »
  4. Donnez un nom et tapotez Ajouter
  5. L'icône apparaît sur votre écran !

Android — Chrome

  1. Ouvrez Chrome et naviguez sur le site
  2. Tapotez le menu ⋮ en haut à droite
  3. Tapotez « Ajouter à l'écran d'accueil »
  4. Confirmez en tapotant Ajouter
  5. L'application s'ouvre comme une appli native !

PC / Mac — Edge ou Chrome

  1. Ouvrez Edge ou Chrome
  2. Cliquez sur l'icône dans la barre d'adresse (ou Menu → Installer l'application)
  3. Cliquez sur « Installer »
  4. Un raccourci apparaît dans vos applications !
Avantage PWA : Pas de téléchargement. Fonctionne sur tous les appareils. Mises à jour automatiques. Expérience similaire à une application native.

Application Web Progressive (PWA)

Notre site est une PWA : installez-le sur n'importe quel appareil depuis votre navigateur, sans passer par un store. Fonctionne hors connexion, se met à jour automatiquement.
